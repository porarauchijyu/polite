# 実装計画：RSS Generator Utility GUI 化 & ビジュアルセレクタ追加

## 目的
既存の C# コンソールアプリケーションを、リッチなデザインの WPF アプリケーションへ移行します。これにより、サイトを見ながら直感的に抽出対象を指定できる「ビジュアルセレクタ」機能を実現し、自動推論が不十分なサイトでも確実に設定できるようにします。

## ユーザーレビューが必要な項目
> [!IMPORTANT]
> アプリケーションの形態が **CLI から Windows デスクトップアプリ（WPF）** へ完全に移行します。
> これまで通りの CLI 操作（引数による自動巡回など）も維持する予定ですが、メインの操作は GUI で行う形になります。

## 構成案

### 1. デザインシステム (Premium Edition)
- **カラーパレット**: ダークモードベース（深みのある濃紺、スレートグレー、アクセントにシアン）。
- **フォルム**: 角を少し丸めたカードデザイン、適度な余白、ガラス質のエフェクト（Blur）。
- **UI コンポーネント**: グラデーションボタン、マイクロアニメーション付きのリスト。

### 2. 主要機能

#### デスクトップ・ダッシュボード
- **サイトリスト**: カード形式で登録サイトを表示。ステータスが一目でわかる。
- **一括アクション**: 「すべてのサイトを更新」「同期」などの迅速な操作。
- **ログ表示**: 内部の処理状況を下部にリアルタイム表示。

#### ビジュアル・セレクタ (Visual Selector)
- **自動推論**: 新規追加時、まずは既存の `InferenceService` で自動解析を行う。
- **手動選択モード**: 「目で見て選ぶ」ボタン。
- **ブラウザ連携**: PuppeteerSharp を非ヘッドレスモードで起動し、専用の JavaScript ツールを注入。
- **直感的操作**: 要素をホバーすると枠が表示され、クリックでセレクタを確定。

## 質問事項
