# RSSジェネレーター・ユーティリティ 活用ガイド

このツールは、ウェブサイトの新着情報を自動的に取得し、RSSフィードとして集約するための高機能ユーティリティです。

## 🚀 基本的な使い方

最もシンプルな使い方は、プログラム（`RssGenerator.exe`）をダブルクリックして実行することです。

1. **実行**: `RssGenerator.exe` を実行します。
2. **自動巡回**: `targets.json` に登録されている全てのサイトを順番にチェックします。
3. **フィード生成**: 新着記事が見つかると、プロジェクトフォルダ内に `feed.xml` が生成（または更新）されます。

---

## 🔍 新しいサイトを追加する（自動推論）

手動で設定を書く必要はありません。コマンドプロンプトや PowerShell から以下のコマンドを実行するだけで追加できます。

```powershell
.\RssGenerator.exe --add [追加したいサイトのURL]
```

**例:**
```powershell
.\RssGenerator.exe --add https://www.westjr.co.jp/press/
```

- アプリがサイトの構造を解析し、最適なセレクタを提案します。
- 画面の案内に従って `y` を入力すれば、自動的に設定が保存されます。

---

## ⚙️ 詳細な設定 (`targets.json`)

`targets.json` を直接編集することで、監視対象を細かく管理できます。

```json
{
  "Url": "サイトのURL",
  "TitleSelector": "記事タイトルを特定するCSSセレクタ",
  "LinkSelector": "記事リンクを特定するCSSセレクタ"
}
```
※ **UPSERT機能**: ファイルを保存してアプリを再実行すると、データベース内の設定も自動的に同期・更新されます。

---

## 📂 生成されるファイル

- **`feed.xml`**: 生成されたRSSフィードです。RSSリーダーなどにこのファイルを登録して使用します。
- **`feed.sqlite`**: 過去に取得した記事を記録しているデータベースです。
- **`targets.json`**: 監視対象のリストです。

## 💡 ヒント
- **初回実行時**: ブラウザ（Chromium）の自動ダウンロードが始まることがありますが、しばらく待つと完了します。
- **巡回間隔**: タスクスケジューラなどで定期的に実行するように設定すると便利です。
