# サイト別 RSS フィード生成機能の実装

現在の全体まとめとしての `feed.xml` に加え、登録されているサイトごとに個別の RSS フィードファイルを生成・保存する機能を実装します。

## ユーザーレビューが必要な項目

> [!IMPORTANT]
> **保存場所とファイル名**
> - 個別フィードは、実行ファイルと同じディレクトリの `feeds` フォルダ内に保存するようにします。
> - ファイル名は `feed_{サイト名}.xml` とします（サイト名にファイル名として無効な文字が含まれる場合は置換します）。
