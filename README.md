# Polite RSS Feeds

GitHub Actions を使用して自動生成されている RSS フィードの一覧です。
リポジトリがパブリックになると、以下の URL を RSS リーダー（Feedly, Inoreader 等）に登録して利用できます。

## 統合フィード
すべての更新をまとめたフィードです。

- **[全サイトまとめフィード](https://raw.githubusercontent.com/porarauchijyu/polite/main/feed.xml)**
